Installation Guide

Install Proxmox VE
from scratch

Bare-metal hypervisor setup in under 20 minutes. Follow the steps below — each section includes the exact commands you need.

8 steps total
~15 min setup time
Proxmox VE 8.x
01

Download the ISO

Grab the latest Proxmox VE ISO from the official site. You'll need a USB drive (≥2GB) or a tool like Balena Etcher / Rufus to flash it.

required
bash 
# Download via wget (or grab from https://www.proxmox.com/downloads)
wget https://enterprise.proxmox.com/iso/proxmox-ve_8.2-1.iso
Tip Flash the ISO with: dd if=proxmox-ve_8.2-1.iso of=/dev/sdX bs=1M status=progress — replace /dev/sdX with your USB drive.
02

Boot from USB

Insert the USB into your target machine. Enter the BIOS/UEFI boot menu (usually F2, F11, F12 or DEL) and boot from the USB drive.

Note Make sure Virtualization (VT-x / AMD-V) is enabled in BIOS. Proxmox requires hardware virtualization support.
03

Run the Installer

Select "Install Proxmox VE (Graphical)" from the boot menu. Accept the EULA and choose your target disk for installation.

Warning The target disk will be wiped. Pick the right drive. ZFS RAID options are available here if you have multiple disks.
04

Set Location & Timezone

Choose your country, timezone, and keyboard layout. These affect locale settings and the system clock.

05

Set root Password & Email

Enter a strong root password — this is your admin password for the web UI and SSH. Email is used for cluster alerts (optional but recommended).

06

Configure Network

Set your hostname, IP address, gateway, and DNS. Use a static IP — you don't want Proxmox's address changing.

example values 
Hostname:    pve.local
IP Address:  192.168.1.100/24
Gateway:     192.168.1.1
DNS:         1.1.1.1
07

Install & Reboot

Review the summary and click Install. The process takes 3–5 minutes. Remove the USB when prompted, then reboot.

Tip After reboot you'll see the console with the web UI address. Open it in your browser: https://<IP>:8006
08

Post-Install Setup

Log into the web UI at https://<IP>:8006 as root. Run these commands over SSH to disable the subscription nag and update the system.

recommended
ssh — disable nag & update 
# Disable enterprise repo (requires subscription)
sed -i 's/^deb/#deb/' /etc/apt/sources.list.d/pve-enterprise.list

# Add free community repo
echo "deb http://download.proxmox.com/debian/pve bookworm pve-no-subscription" \
  > /etc/apt/sources.list.d/pve-community.list

# Update system
apt-get update && apt-get -y upgrade
bash — remove subscription nag popup 
sed -Ezi.bak "s/(Ext.Msg.show\(\{[\s\S]*?title: gettext\('No valid sub)/void(\1/g" \
  /usr/share/javascript/proxmox-widget-toolkit/proxmoxlib.js

# Restart the web service
systemctl restart pveproxy
Done Proxmox VE is ready. Access the dashboard at https://<your-ip>:8006 and start creating VMs and containers.
