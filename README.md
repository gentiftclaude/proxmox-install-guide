# Proxmox VE Install Guide

A clean, single-page installation guide for **Proxmox VE 8.x** — covers everything from downloading the ISO to post-install setup.

## Live Site

[proxmoxinstall.gentjusufi01.xyz](https://proxmoxinstall.gentjusufi01.xyz)

## What's Covered

1. Download the ISO
2. Boot from USB
3. Run the installer
4. Set location & timezone
5. Set root password & email
6. Configure network (static IP)
7. Install & reboot
8. Post-install setup (disable nag, add community repo, update)

## Tech

- Pure HTML/CSS/JS — no frameworks, no dependencies
- JetBrains Mono + Syne fonts via Google Fonts
- Copy-to-clipboard on all code blocks

## Usage

Just open `index.html` in a browser — no build step needed.
